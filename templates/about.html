About CryptoTracker BMZ

A cryptocurrency price tracking web application built with Flask, featuring real-time market data from CoinGecko API, user authentication with admin approval system, watchlist management, and machine learning-based price analysis using 4 indicators.

Features

  • Live Market Data — Real-time cryptocurrency prices from CoinGecko API with 2-minute cache
  • Market Overview — Total market cap, 24h volume, BTC dominance, active cryptocurrencies
  • Trending Coins — Top 7 trending cryptocurrencies from CoinGecko
  • Price Charts — Interactive 7D / 30D / 90D price history charts with SMA-7, SMA-30, Bollinger Bands overlays

  • Machine Learning Signals — 4 indicators: Linear Regression, RSI, SMA Crossover, Bollinger Bands
  • Overall Signal — Combined BUY/SELL/HOLD/STRONG BUY/STRONG SELL with confidence score
  • Dark / Light Mode — Toggle theme with session persistence

Machine Learning — 4 Indicators

This app uses 4 ML indicators to analyze cryptocurrency price trends and generate trading signals.

1. Linear Regression — 7-Day Price Prediction

Uses Simple Linear Regression to forecast prices 7 days ahead based on 30 days of historical data.

  1. Data Collection — Collects 30 days of daily price history from CoinGecko API
  2. Model Training — Fits a Linear Regression model: y = mx + c
    • x = day index (0, 1, 2, ..., 29)
    • y = price in USD
  3. Prediction — Forecasts price at day n + 7 (7 days ahead)
  4. Signal Generation — Classifies as STRONG BUY / BUY / HOLD / SELL / STRONG SELL based on predicted change %
  5. Confidence Level — Based on R² score (coefficient of determination)

2. RSI (Relative Strength Index)

Momentum oscillator measuring speed and change of price movements (0-100).

RSI ValueInterpretationSignal
RSI > 70Overbought — potential sell signalSELL
RSI < 30Oversold — potential buy signalBUY
30 - 70Neutral zoneHOLD

3. SMA Crossover (Simple Moving Average)

Compares short-term vs long-term trend using two SMAs:

  • SMA-7 — 7-day simple moving average (short-term)
  • SMA-30 — 30-day simple moving average (long-term)
ConditionSignal
SMA-7 > SMA-30 (golden cross)BULLISH
SMA-7 < SMA-30 (death cross)BEARISH

4. Bollinger Bands

Volatility bands placed above and below a moving average (middle band = 20-day SMA).

Price PositionSignal
Price near lower bandBUY (oversold)
Price near middle bandHOLD
Price near upper bandSELL (overbought)

Technologies Used

  • Backend: Flask 3.0+, Python
  • Database: SQLite with Flask-SQLAlchemy
  • Auth: Flask-Login, Werkzeug password hashing PBKDF2-SHA256
  • ML: scikit-learn (LinearRegression), NumPy
  • API: CoinGecko (free tier + multi-key fallback)
  • Charts: Chart.js 4.4 (CDN)
  • Icons: Lucide Icons (SVG)
  • CSS: Custom CSS with CSS Variables (Dark/Light themes)

Limitations & Disclaimer

Important: This prediction is for educational/demonstration purposes only.

  • Linear Regression assumes a linear relationship — crypto markets are highly non-linear
  • The model does not incorporate external factors (news, sentiment, macroeconomic data)
  • Cryptocurrency markets are extremely volatile and unpredictable
  • Past performance does NOT guarantee future results
  • Do NOT use this prediction for actual investment decisions
