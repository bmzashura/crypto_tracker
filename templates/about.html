About CryptoTracker

Tema Website

CryptoTracker adalah website monitoring harga cryptocurrency berbasis Flask yang mengintegrasikan data real-time dari CoinGecko API. Website ini dirancang dengan desain terinspirasi dari Kraken Exchange menggunakan sistem warna ungu dan hijau khas fintech crypto.

API yang Digunakan

CoinGecko API — Gratis, reliable, dan menyediakan data untuk 18,000+ cryptocurrency dari 1,000+ exchange.

  • Free tier: 10,000 calls/month
  • Data: Price, Market Cap, Volume, Supply, ATH/ATL, dll
  • No authentication required (public endpoints)

Cara Kerja Pengambilan Data API

Data diambil melalui proses berikut:

  1. Flask route memanggil CoinGecko API endpoint /coins/markets
  2. Request menggunakan library requests dengan API key di header
  3. Response dalam format JSON langsung di-parse oleh Python
  4. Data divisualisasikan menggunakan template HTML/Jinja2

Fitur Utama

  • Daftar Coin — Menampilkan 20 crypto teratas berdasarkan market cap
  • Pencarian — Cari coin berdasarkan nama atau simbol
  • Detail Coin — Halaman detail untuk setiap cryptocurrency
  • Filter & Sort — Navigasi paginasi untuk melihat lebih banyak data
  • Desain Responsif — Tampilan optimal di desktop dan mobile

Insight & Manfaat

Website ini bermanfaat untuk:

  • Edukasi — Memahami cara kerja API dalam web development
  • Monitoring — Melihat harga crypto secara real-time tanpa install aplikasi
  • Referensi — Contoh implementasi Flask + External API integration
