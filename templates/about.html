Machine Learning -- Linear Regression Price Prediction

This app uses Simple Linear Regression to forecast cryptocurrency prices based on historical data.

How It Works

  1. Data Collection -- Collects 30 days of daily price history from CoinGecko API
  2. Model Training -- Fits a Linear Regression model: y = mx + c
    • x = day index (0, 1, 2, ..., 29)
    • y = price in USD
  3. Prediction -- Forecasts price at day n + 7 (7 days ahead)
  4. Signal Generation -- Classifies as STRONG BUY / BUY / HOLD / SELL / STRONG SELL based on predicted change %
  5. Confidence Level -- Based on R^2 score (coefficient of determination)

Note: If R^2 < 0.3, signal is always HOLD (low confidence).

Confidence Levels (R^2 Score)

R^2 ScoreConfidenceMeaning
> 0.7HighModel fits data very well
0.4 - 0.7MediumModerate fit
< 0.3LowSignal forced to HOLD

R^2 Score Formula

R^2 = 1 - (SS_res / SS_tot)

where:
  SS_res = sum((y_actual - y_predicted)^2)  (residual sum of squares)
  SS_tot = sum((y_actual - y_mean)^2)        (total sum of squares)

Limitations & Disclaimer

Important: This prediction is for educational/demonstration purposes only.

  • Linear Regression assumes a linear relationship -- crypto markets are highly non-linear
  • The model does not incorporate external factors (news, sentiment, macroeconomic data)
  • Cryptocurrency markets are extremely volatile and unpredictable
  • Past performance does NOT guarantee future results
  • Do NOT use this prediction for actual investment decisions
